TranslateX()

The translateX() function moves an element horizontally along the X-axis.
A positive value shifts it to the right, and a negative value shifts it to the left.

Code: transform: translateX(100px);

TranslateY()

The translateY() function moves an element vertically along the Y-axis.
A positive value moves it downward, and a negative value moves it upward.

Code: transform: translateY(100px);

Translate()

The translate() function moves an element in both horizontal and vertical directions.
The first value is for the X-axis (left/right), and the second is for the Y-axis (up/down).

Code: transform: translate(100px, 50px);

ScaleX()

The scaleX() function resizes an element horizontally.

Code: transform: scaleX(1.5);

ScaleY()

The scaleY() function resizes an element vertically.

Code: transform: scaleY(1.5);

Scale()

The scale() function resizes an element in both X and Y directions.

Code: transform: scale(1.5, 1.2);

Rotate()

The rotate() function rotates the element in 2D space (Z-axis).
Positive values rotate clockwise, negative values counter-clockwise.

Code: transform: rotate(45deg);

RotateX()

The rotateX() function rotates the element around the X-axis (horizontal axis).
This creates a vertical 3D flip effect.

Code: transform: rotateX(45deg);

RotateY()

The rotateY() function rotates the element around the Y-axis (vertical axis).
This creates a horizontal 3D flip effect.

Code: transform: rotateY(45deg);

SkewX()

The skewX() function skews an element horizontally.

Code: transform: skewX(20deg);

SkewY()

The skewY() function skews an element vertically.

Code: transform: skewY(20deg);

Skew()

The skew() function skews an element in both directions.

Code: transform: skew(20deg, 10deg);
